03How I work

Five things you should know before hiring me.

1. I answer the phone. Until I can't.

When I'm on another job, the AI agent picks up and books you in. Every client of mine who calls knows exactly which is which — the agent says "Hyde Tech, I'm Shawn's AI agent" up front. No pretending.

2. Flat quotes on anything I can scope.

Site survey, 4-camera install, WiFi mesh, AI agent build — all flat-rate, quoted in writing before any work starts. Hourly only on things I genuinely can't scope in advance, and I'll tell you that's why.

3. No contracts longer than a month.

Managed services run month-to-month. If I'm not earning my keep you can cancel in 30 days. I don't believe in holding clients hostage to annual agreements.

4. I'll tell you if it's not a fit.

I'm not the right fit for enterprise IT, SOC-2 compliance audits, or 24/7 human NOC coverage. If that's what you need, I'll say so and point you somewhere better.

5. I use the stuff I sell.

The AI phone agent on (289) 512-0207 is the same platform I'd set up for you. The cameras in my own office are Reolink. My network is UniFi. If I don't use it, I don't sell it.
