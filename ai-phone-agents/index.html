06Honest questions

Stuff you're probably wondering.

Does it sound like a robot?

It sounds like a person. Call (289) 512-0207 and judge for yourself — we picked the platform partly because it doesn't sound like one of those robocall IVRs.

What happens if it gets something wrong?

Every transcript gets flagged if the agent fumbles. We review them weekly the first month, monthly after. When we find a miss, we update the script and the next caller gets the right answer.

Can I take over a call while it's happening?

Yes — we can configure warm-transfer to your cell. For certain call types (VIP customers, complaints, insurance claims) the agent will ring you directly instead of trying to handle it.

What if I already use Jobber or Housecall Pro?

Supported. The agent books directly into your scheduling system using their API. You see the booking in the same place as everything else.

Do you keep recordings?

Transcripts, yes — for 12 months, so you can search them. Audio recordings, only if you opt in and notify callers (Canadian two-party-consent rules). Default is transcript-only.

Can I cancel?

Month-to-month after setup. 30 days' notice, no long-term contract. We'll even port your number back to your old provider if you leave — no hostage-taking.
